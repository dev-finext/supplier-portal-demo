# פורטל ספקים — דמו פרונטאנד

מערכת הדגמה לפורטל ספקים: כניסה עם טלפון + קוד חד־פעמי, ניהול הזמנות (ממתינות / מאושרות / מבוטלות), אישור טיפול בודד ומרוכז, סינון לפי תאריכים, ייצוא לאקסל וכרטיס הזמנה מלא.

**פרונטאנד בלבד — ללא בקאנד.** כל הנתונים הם נתוני דמו סטטיים בצד הלקוח, לצורכי הדגמה למפתחים וללקוחות עסקיים. שום מידע לא נשלח לשרת.

## מבנה

- `index.html` — כל מסכי האפליקציה (התחברות, אימות, פאנל הזמנות, מודאלים)
- `css/styles.css` — עיצוב מלא כולל גרסת מובייל רספונסיבית
- `js/data.js` — נתוני הדמו (ספק + הזמנות)
- `js/app.js` — ניהול מצב, רינדור ואירועים (Vanilla JS, ללא תלויות)

## הרצה מקומית

לפתוח את `index.html` בדפדפן. אין צורך בשרת או בהתקנות.

## פריסה (CI/CD)

כל push ל־`main` מפעיל GitHub Actions שמפרסם אוטומטית ל־GitHub Pages — הקישור נשאר קבוע.

## הערות דמו

- מסך ההתחברות מקבל כל מספר טלפון וכל קוד — אין אימות אמיתי.
- "ייצוא לאקסל" מוריד קובץ CSV אמיתי (UTF-8 עם BOM) לפי הלשונית והסינון הפעילים.
- אישורי טיפול נשמרים בזיכרון הדפדפן בלבד ומתאפסים ברענון.
